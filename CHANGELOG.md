## [1.2.0] - 2026-04-30

### Added
- 7 LATAM data products (App Downloads, Closeup Prescriptions, Discount Program Activity, Lilly360, WhatsApp, Lilly Online Store, SOA)
- Storage & Access section showing `Lives In` and `Table Name` per contract
- "All Locations" filter dropdown (IRIS / SFMC DE / S3 Refined Bucket)

### Changed
- Header revised to "Data Inventory" / "IBU Omnichannel Current State Data Contracts"
- Contract count: 25 → 32

## [1.1.0] - 2026-04-29

### Added
- 25 curated data products replacing the previous 50-sheet view
- Multi-table support (Customer Data Processing shows both `Master_Customer_Data` and `Master_Profile_Data`)
- Veeva region-flag matrix rendering for `sent_email`, `account`, `Call2_vod__c`

## [1.0.0] - 2026-04-29

### Added
- Initial Data Contracts Repository dashboard
- Master-detail layout with searchable / filterable contract list
- Schema viewer with column type, description, example
- Lilly logo branding in header and footer
